The main hub for all of my random coding projects.

Space.js (Space Game)

My latest project, and still being updated (on hiatus while working on framework): A game about space! Quite original, isn't it? The idea is that you are given a randomly generated solar system to fly around in, hopefully with some enemies and other AI to interact with in the future.

Latest Changes:

  • 3/12/19 Restructured the code, fixed moons orbiting moons, and added another ship with no AI.

Wynd.js (Island Game)

My foray into Perlin Noise generation. This game generates a random island cloaked in black for you to uncover and map out as you play. It regenerates the island each reload.

Cerijan

A blast to the past! This is the first game I made. Very lightweight and grindy, this game sports only about five different types of enemies, a rudimentary level up system, and two purchaseable items.

Latest Changes:

  • 3/12/19 Fixed a four year old bug where you could drink more potions than you had.
  • 3/13/19 Fixed another four year old bug where the interaction buttons would not change from the "Leave" or "Accept" options for buying something.

Welcome to Nightvale Encyclopedia

An encyclopedia for the podcast Welcome to Nightvale. It was my little attempt at HTTP requests, so that I could easily add entries with only text files.

Prop.js Framework

My biggest project: a heavily object-oriented framework to manage and draw objects for games. The framework itself is small, but it can be expanded through the application of modules.

Old Khan Academy Stuff

My old Khan Academy account from when I started programming at the age of 10. Money Bag was the first game I made, and Falling Stars was a remake of that. This code is extremely messy, but I kept it for nostalgia and posterity.
